1. Executive Summary

Innovate is a B2B digital creative agency (Astro v6.2.2, Vercel) targeting tech startups, financial services, and luxury brands. The full audit covered 5 pages across 9 specialist subagents. The site scores 31/100 — in the Critical range, primarily because the entire metadata and structured data layer is missing, not because the technical foundation is broken.

Top 3 Critical Issues:
  1. No robots.txt (404) — AI crawlers and Googlebot operate with zero directives
  2. No canonical tags on any page — Vercel preview URLs may split crawl equity
  3. Zero structured data sitewide — no Knowledge Panel eligibility, no rich results
What's working well:
  • Astro SSR: all content in HTML at crawl time — Googlebot doesn't need JavaScript
  • HTTPS enforced with strong HSTS (2-year max-age, includeSubDomains, preload)
  • Clean URL structure — all pages accessible within 1 click from home
  • Good TTFB (~219ms) from Vercel Singapore CDN edge

11. Prioritized Action Plan

CRITICAL Fix Immediately

C-1. Create /public/robots.txt Effort: 30 min

No robots.txt at /robots.txt (404). All AI crawlers (GPTBot, ClaudeBot, Google-Extended, PerplexityBot) and Googlebot operate with zero directives. No Sitemap pointer. Impact: Crawlability, GEO, AI indexing.

C-2. Add canonical tags to Layout.astro Effort: 5 min

Zero canonical tags sitewide. Vercel preview URLs can index alongside the primary domain, splitting crawl equity. Add <link rel="canonical" href={Astro.url.href} /> to Layout.astro head.

C-3. Write unique meta descriptions for all 5 pages Effort: 1 hr

All pages use "Premium Corporate Agency" (25 chars). Google will auto-generate snippets. Target 140–160 characters per page with service keywords. See Section 3 for recommended text.

C-4. Add Organization + WebSite + FAQPage JSON-LD to homepage Effort: 2 hr

Zero schema on all 5 pages. The homepage FAQ section qualifies for AI Overview citation with FAQPage markup. Organization schema enables Knowledge Panel eligibility. Full code in Section 12.

C-5. Add Open Graph and Twitter Card meta tags Effort: 30 min

No OG or Twitter Card tags on any page. LinkedIn, Slack, and WhatsApp shares show blank previews. For a design agency, blank social previews are a brand-damaging gap. See Section 4 for code.

C-6. Fix all href="#" placeholder links Effort: 30 min

Privacy Policy, Terms of Service, LinkedIn, Twitter, and Instagram all link to "#". Privacy Policy / Terms are a legal compliance gap. Replace with real URLs or remove before launch.

HIGH Fix Within 1 Week

H-1. Fix FadeUp LCP issue — remove opacity:0 from SSR HTML Effort: 2–3 hr

Hero H1 renders invisible in SSR HTML until JS hydration. This is the primary mobile LCP regression (+1,500–2,700ms). Replace React FadeUp island with CSS-only @keyframes animation.

H-2. Lazy-load images + fix responsive sizing Effort: 2 hr

All 7 images load eagerly at 2,070px width regardless of device. ~1.7 MB excess on mobile. Use Astro's <Image> component with loading="lazy", fetchpriority="high" on first image, and w=800 srcset. Fix broken Zenith App 404 image.

H-3. Move GSAP + Lenis to client:idle Effort: 1 hr

GSAP (111 KB) loads in the critical path on every page. Causes ~400ms TBT on mobile. Change client:load to client:idle for animation libraries. Convert CustomCursor and FadeUp to vanilla JS.

H-4. Strengthen title tags on all pages Effort: 15 min

Current titles are generic ("Our Services | Innovate"). See Section 4 for keyword-targeted alternatives. Highest CTR impact per effort ratio of any fix on the site.

H-5. Add named team members to /about page Effort: 1–2 hr

No named individuals anywhere on the site. Google's E-E-A-T framework and AI citation systems both require named, verifiable principals. Add founder name + LinkedIn, 2–3 team profiles with roles and photos.

H-6. Add metrics to portfolio /work entries Effort: 2 hr

Six portfolio entries show only name, category, and year. Zero quantified outcomes. Add at least one metric per project. B2B buyers compare agencies on proven results, not aesthetics alone.

H-7. Create /public/llms.txt Effort: 1 hr

Emerging standard for AI discovery. Checked by Perplexity, ChatGPT Browse, and Claude.ai before deep crawling. See Section 7 for recommended content.

H-8. Add security headers to vercel.json Effort: 30 min

Missing: X-Content-Type-Options, X-Frame-Options, Referrer-Policy, Content-Security-Policy. See Section 2 for vercel.json configuration.

MEDIUM Fix Within 1 Month

M-1. Fix site property in astro.config.mjs Effort: 5 min

@astrojs/sitemap integration is configured but won't build without the site property. Add: site: 'https://innovate-proto.vercel.app'

M-2. Add width/height to all img tags + fix alt text Effort: 1 hr

No image dimensions → CLS. Truncated alt text (stops at "&") → image SEO loss. Use Astro's <Image> component for automatic sizing, add full descriptive alt text per project.

M-3. Service schema for /services page Effort: 2 hr

4 service pillars (Brand Identity, Digital Experience, Technical Execution, Strategic Growth) each need Service schema with sub-service OfferCatalog. Full code in Section 12.

M-4. Add aria-expanded to mobile hamburger button Effort: 15 min

Missing aria-expanded attribute is a WCAG 2.1 AA violation. Toggle true/false on open/close.

M-5. Self-host Google Fonts Effort: 30 min

External Google Fonts is a render-blocking stylesheet. Use @fontsource/manrope and @fontsource/inter packages. Eliminates external round-trip, improves FCP by 3–5 pts.

M-6. Register branded custom domain before any SEO investment Effort: 1 hr

innovate-proto.vercel.app is a staging domain. No link equity can be built. Register custom domain, configure 301 redirect from Vercel, then begin all SEO work on the production URL.

LOW Backlog

  • Implement IndexNow via Vercel's native deploy integration (instant Bing/Yandex indexing on deploy)
  • Add apple-touch-icon.png and favicon-32x32.png alongside existing favicon.svg
  • Add s-maxage=86400 stale-while-revalidate to HTML Cache-Control in vercel.json
  • Disable Lenis smooth scroll for prefers-reduced-motion users
